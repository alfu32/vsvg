# SVG rendering context

web - Canvas like rendering context 2d

What's working:
- (poly)lines (begin(),stroke(),fill(),line_to,arc_to)
- rectangle
- ellipse
- stroke_text
- fill_text
- context save and restore
- context transforms (translate, rotate, scale)
- context parameters:
  - fill_style
  - stroke_style
  - line_cap
  - line_dash_offset
  - line_join
  - line_width
  - miter_limit
  - direction
  - font
  - font_kerning
  - text_align
  - text_baseline
  - line_dash
